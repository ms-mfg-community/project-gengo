# Simple Calculator

A Java command-line calculator application that performs basic arithmetic operations.

## Features

- Addition, subtraction, multiplication, and division operations
- Interactive user interface
- Input validation and error handling
- User-friendly prompts and messages

## Requirements

- Java 11 or higher
- Maven 3.6 or higher
