## Building the Project

### Prerequisites
- C++17 compatible compiler
- CMake (version 3.14 or higher)
- Git

### Build Instructions
```bash
# Clone the repository
git clone <repository-url>
cd thermostat

# Create build directory
mkdir build && cd build

# Configure with CMake
cmake ..

# Build
cmake --build .
```

### Running Tests
```bash
cd build
ctest
```
