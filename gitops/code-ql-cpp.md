# CodeQL C/C++
\n\nBuild scanner action

Create a CodeQL GitHub workflow action to scan C/C++ code in this repository on pushes or PRs, as well as schedule to run every monday at 02:00. Create a corresponding CMakeLists.txt file in the root of this repo for it also.
\n\nNOTE

\_To User: As a pre-requisite, code scanning must be set to 'Advanced' instead of 'Default' for the targeteted repository.
